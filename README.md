# 📂 File Organizer (Python)

A Python script that automatically organizes files into categorized folders like Images, Videos, Documents, Music, and more.

## 🚀 Features
- Automatically sorts files by type
- Creates folders dynamically
- Handles unknown files (AND MOVE -> Others folder)
- Takes user input for folder path

## 🛠️ Tech Used
- Python
- os module
- shutil module

## ▶️ How to Run
1. Run the script:
   python organizer.py
2. Enter the folder path when prompted

## 📌 Future Improvements
- GUI version
- Duplicate file detection
- File size-based sorting
